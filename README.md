## Installation

Install std by adding it as a [rebar](https://github.com/basho/rebar)
dependency:

```erlang
% In your rebar.config:
{deps, [
    % ...
    std, ".*", {git, "git://github.com/johannesh/estd.git"}}
    % ...
]}.
```

Then run `rebar get-deps` to sync your dependencies.
